# ✨ Glassmorphic Signup Page

A modern and responsive **Signup Page UI** built using **HTML & CSS** with a beautiful **glassmorphism effect**.

---

## 🚀 Features

* 🔮 Glassmorphism UI design (blur + transparency)
* 📱 Responsive layout
* 🎨 Clean and minimal interface
* 🔐 Input fields:

  * Name
  * Email
  * Password
* ✅ "Create Account" button
* 🔗 Login link for existing users
* 🧾 Signup heading at the top

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Glassmorphism effects)

---

## 📂 Project Structure

```
signup-page/

```
